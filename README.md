# 🌐 Vulnerability Assessment Report — future_CS_01

## 📌 Project Overview
This project demonstrates a basic vulnerability assessment conducted on a live website using industry-standard tools.

The goal was to identify common web security issues, classify risks, and provide remediation advice in clear business language.

---

## 🎯 Scope of Assessment
Target Website: https://testphp.vulnweb.com  
Assessment Type: Non-intrusive / Passive Security Testing  
Testing Methodology: Reconnaissance + Passive Analysis

⚠️ Only publicly accessible resources were tested.  
No exploitation or intrusive testing was performed.

---

## 🛠 Tools Used
• Nmap — Network scanning & service discovery  
• OWASP ZAP (Passive Scan) — Web vulnerability detection  
• Browser DevTools — Client-side security inspection  
• Canva — Professional report design

---

## 🔎 Skills Demonstrated
• Vulnerability Analysis  
• Risk Classification (Low / Medium / High)  
• Security Reporting  
• Client-friendly Communication
